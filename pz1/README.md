## Примітка щодо Packet Tracer

На комп'ютері знайдено профіль Packet Tracer 7.3.0, але виконуваний файл `PacketTracer.exe` відсутній у стандартних шляхах встановлення. Тому в папці підготовлені матеріали, схеми та звіт із кодом; нативні `.pkt` файли потрібно зберегти вручну після вставлення відповідного коду в Cisco Packet Tracer.
